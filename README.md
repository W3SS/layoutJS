layoutJS
========

layoutJS is Javascript Library to help you layout your content,
the documentation is coming soon. ;)

example:

<!-- VIEW -->
<div data-layout="vertical">
  <div data-layout-constraints="expand"> content1 </div>
  <div style="height:50%;"> content1 </div>
  <div data-layout-constraints="expand"> content1 </div>
</div>
